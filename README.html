Applied Probability for Computer Science Notes¶

These are my notes from the Master’s Degree Programme course [CM0546] Applied Probability for Computer Science by Prof. Antoniano Villalobos.

Disclaimer¶

As these are my personal notes I might have misunderstood or came up to wrong conclusions during the lessons. For these mistakes I apologize in advance and incourage anyone to submit either a GitHub issue on this repository so that I will fix the mistake as soon as possible or submit directly a pull request.

The course content is a selection from the textbook Probability and Statistics for Computer Scientists of which I’m not the author.

Technologies¶

Anaconda¶

Personally I install most of my Python packages with Anaconda which I also use to manage development environments.

Jupyter Notebooks¶

Jupyter is an open source platform to create interactive software, usually in the form of notebooks, in a wide variety of programming languages.

Installing Jupyter¶

conda install -c conda-forge jupyter

JupyterBook¶

This is a neat conversion and pagination tool for Jupyter and Markdown books (and notes).

Installing JupyterBook¶

At the time of writing JupyterBook is not installable through Anaconda but only through PyPI.

pip install jupyter-book

Building a PDF for the notes¶

jupyter-book build <course folder path> --builder pdfhtml

Cleaning the build outputs¶

jupyter-book clean <course folder path>

IRKernel¶

This is a native R kernel for Jupyter. It is part of the r-essentials package in Anaconda.

Setting up R with Anaconda¶

conda install -c r-essentials r-base
